The global task management market is valued at **$4.1B** (2025) and growing at 13% CAGR. Yet every major player — Todoist, Notion, TickTick, Google Tasks, Asana — shares the same fundamental limitation:

> **They are passive filing cabinets.** Users manually create, categorize, schedule, and track tasks. The app does nothing proactive.

**Sentinel Bridge is different.** It's the first **AI-native autonomous task manager** that actively ingests tasks from calendars and documents, auto-categorizes them using machine learning, suggests optimal scheduling using real calendar availability, and self-heals when systems fail.

## Feature Deep Dive — What Makes Sentinel Unmatchable

### 🧠 1. Smart Scheduling — No Competitor Has This

When creating a task, Sentinel reads the user's **actual Google Calendar availability** across all accounts and shows clickable time suggestions:

> *"You have 6 free slots today. Tap 'Today 9:00 AM - 10:00 AM' to schedule."*

**Why it matters:** Every other app forces users to manually pick a time. Sentinel knows when you're free and schedules accordingly. This alone is a category-defining feature.

---

### 📄 2. Document → Tasks Pipeline

Upload a PDF contract, a meeting agenda, or any document. Sentinel's AI extracts all actionable items with due dates, priorities, and categories — automatically.

> *"FDA Food Safety Audit" extracted from uploaded compliance document → categorized as "Legal" → synced to Personal calendar → notification sent.*

**Why it matters:** Notion has databases but no AI extraction. Todoist has email-to-task but no document parsing.

---

### 🤖 3. AI Auto-Categorization with Machine Learning

Sentinel doesn't just categorize — it **learns** from user corrections. Reassign a task's category and the ML engine updates its keyword library for future accuracy.

- 53 keywords across 5 categories (Work, AI, Family, Leo's School, Legal)
- 5 user override corrections learned and applied
- Category → Calendar mapping (Work → work calendar, Family → personal calendar)

---

### 📅 4. Multi-Calendar Write-Back

Sentinel bridges **multiple Google Calendar accounts** — tasks created in Sentinel automatically appear in the right calendar:

| Category | Calendar |
|----------|----------|
| Work, AI | mpetrini@heinleinfoodsusa.com |
| Family, Leo's School | mauro@gelatopetrini.com |

When a task is marked done, the calendar event is automatically deleted. When a task is snoozed, the event updates. **No competitor offers bidirectional multi-account sync.**

---

### 🛡️ 5. Self-Healing Architecture

The system monitors its own health and auto-repairs failures:
- **Auto Token Refresh** — Google OAuth tokens never expire
- **Quiet Hours** — No notifications between 10 PM and 7 AM (except high priority)
- **Exponential Backoff** — Failed API calls retry with 2s → 4s → 8s delays
- **Proactive Architect** — AI-powered system audit detects architectural gaps

---

### 🎤 6. Voice Input

Hold the microphone button → speak naturally → Sentinel uses the Web Speech API to transcribe and Gemini to extract intent, time, and category from speech. Zero typing needed.
