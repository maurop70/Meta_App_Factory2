# C-Suite Disciplinary Rule (CPG/Venture Validation)

## ENFORCEMENT

Any output produced without this confirmation is invalid and must be regenerated.

## Rules
Whenever executing or planning a C-Suite Venture Mode strategic review:
1. **No Speculation**: C-Suite sub-agents are strictly prohibited from using high-level assumptions. They must produce and validate bottom-up calculations.
2. **CFO Mandate**: The CFO Agent must always provide a detailed bottom-up financial model (covering raw material COGS, line-time, labor, packaging, CapEx, and cash-runway) as a deterministic table/spreadsheet. Speculative projections are rejected.
3. **CMO Mandate**: The CMO Agent must always map the competitive landscape (minimum 3 competitors, MSRP, pricing per oz, ingredient profiles, positioning flaws) and provide a line-item tactical marketing budget. Lump-sum budgets are rejected.
4. **CIO Mandate**: The CIO Agent must always provide an Operational & Fulfillment Integration Blueprint mapping storefront-to-ERP APIs, cold-chain thresholds, and carrier selections. Abstract planning is rejected.
5. **Critic Gate**: The Critic must reject any C-Suite output that lacks the required bottom-up structures. Delegation to execution is blocked until the Critic explicitly signs off.
6. **Architect Vetting**: The Architect must stress-test all proposals against these parameters and explicitly state which parameters passed and which failed before approving any plan.
